Overview

WXYZ-Wing uses four cells with candidates forming a restricted pattern. When properly configured, a common candidate can be eliminated from cells seeing all pattern cells.

How It Works

  1. Find four cells whose combined candidates use exactly four digits {W,X,Y,Z}.
  2. One cell (pivot) sees the other three.
  3. Eliminate the restricted digit from cells seeing all four pattern cells.
