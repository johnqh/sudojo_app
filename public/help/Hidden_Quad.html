Overview

A Hidden Quad is the four-cell extension of Hidden Pairs and Triples. When four candidates appear exclusively in four cells within a house, all other candidates can be eliminated from those cells.

How It Works

  1. Within a house, identify four digits that appear only within the same four cells.
  2. These four cells must contain these four digits.
  3. Eliminate all other candidates from these four cells.

Example

Highlighted cells: R7C8, R7C9, R8C8, R8C9

In Block 9, cells R7C8, R7C9, R8C8, R8C9 are the only cells that can contain {1, 2, 5, 7}. These four digits form a Hidden Quad. Eliminate the other candidates {8, 9} from these cells, leaving only the quad digits.

Tips & Patterns

  • Hidden Quads are quite rare - don't spend too much time looking initially.
  • They often appear in puzzles designed to require advanced techniques.
  • Software or systematic checking helps identify them reliably.
