Overview

ALS-Chain links multiple ALS groups through restricted common candidates. When ALS groups share candidates in specific ways, powerful eliminations become possible.

How It Works

  1. Identify multiple ALS groups in the puzzle.
  2. Find restricted common candidates (RCC) that link ALS groups.
  3. An RCC appears in both ALS groups but all instances see each other.
  4. Chain the logic to find eliminations.

Example

Highlighted cells: R4C1, R4C2, R4C5, R4C7, R1C6, R6C6, R1C2, R3C7, R1C8

ALS-Chain pattern involving 9 cells with candidates {2, 3, 4, 5, 7, 8, 9}. Multiple Almost Locked Sets are linked through restricted common candidates, enabling eliminations that single ALS cannot achieve.

Tips & Patterns

  • ALS-Chains require solid understanding of basic ALS.
  • Start with ALS-XZ (simplest chain), then progress to longer chains.
  • These are among the most powerful elimination techniques.
